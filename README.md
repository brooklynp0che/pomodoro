this is a simple pomodoro timer intended to be played on a big screen in the background, as its only cue for breaks are through numbers, not sounds.

i made this since i found myself annoyed by the lack of 40/20 timers online, which is my personal favorite pomodoro method.
also, having the blaring alarms that sounded made it very hard to stay locked in and instead made me feel stressed out

i made this website using primarily css and js, with some html as the base. i made it my favorite colors :)

a main challenege i ran into was figuring out the time logic, but thanks to w3 schools and slackoverflow i was able to figure out my issues and get it working !!

please enjoy the cutie pomodoro i made :)
